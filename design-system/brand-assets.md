## Recommended reuse

- Use `grass-logo.svg` for app header / favicon-level branding.
- Use `ev-lawn-hero.svg` for homepage hero sections or pitch decks.
- Use `grassy-bg.avif` as a softer background texture layer behind bold headline blocks.

## Visual rules

- Prioritize green as atmosphere, not just as a button color.
- Keep hard black borders and offset shadows so the UI still feels like EV.
- Pair calm cream backgrounds with high-contrast black typography.
